# Module 3 – Functions, Math, and User Interaction

## Overview
Focus on basic functions and arithmetic with short, selfcontained scripts.

## Contents
- Area_of_circle.py  compute circle area from radius input.
- F_to_C.py  Fahrenheit to Celsius conversion.
- Greet_only_u&instructor.py  selective greeting logic.
- Hello_World.py  simple hello world.
- MPG_for_Car.py  milespergallon calculator.
- Return_day.py  return or compute a day value from input.
- User_name_greet.py  prompt name and greet user.

## Requirements
- Python 3.8+

## Run
`
python <file_name>.py
`

## Notes
- Great for practicing input validation and printing formatted results.
