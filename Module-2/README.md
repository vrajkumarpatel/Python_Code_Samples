# Module 2 – Numbers, Strings, and Simple Logic

## Overview
This module progresses into numeric operations, simple authentication, Morse code translation, and basic string formatting.

## Contents
- BMI.py  body mass index calculator from user input.
- Hello_World.py  hello world and simple printing.
- Login_Pass.py  credential prompt and simple checks.
- Module 2 Lab Activity.docx  exercise instructions.
- Morse_Code.py  translate letters/numbers to Morse code.
- Name_greet.py  prompt for name, greet with formatting.
- Sum_Product.py  compute sum and product of input values.

## Requirements
- Python 3.8+

## Run
`
python <file_name>.py
`

## Notes
- Each program demonstrates a small concept (loops, conditionals, functions).
- docx file contains the lab writeup; its not executable.
