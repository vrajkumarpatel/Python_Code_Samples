# Module 4  Collections, Branching, and Small Projects

## Overview
Builds toward multifile work and small team assignments. Introduces branching, lists/dicts, and simple modules.

## Contents
- Lab_Activity/ — lab scripts and a README:
  - ranching.py  conditional branching exercises.
  - collection.py  collections (lists, dicts) basics.
  - grading.py  grade calculation logic.
  - pirate.py  themed exercise with input/branching.
  - 	ime.py, 	ime2.py  time handling practice.
- Team_project.py  small team project script.
- uildsandwich.py  simple stepbystep function flow.
- exchange_chart.py  currency or unit exchange chart.
- someprogram.py  general exercise script.
- 	eam_project(new).py  updated team project variant.

## Requirements
- Python 3.8+

## Run
`
python <file_name>.py
`
