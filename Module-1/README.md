# Module 1  Introductory Python Samples

## Overview
This module contains introductory Python programs covering printing output, basic variables, simple conditionals, and input/output.

## Contents
- Average.py  computes the average of provided numbers.
- Firstprog.py  first program demonstration (print and variables).
- Test.py  simple test harness or quick experiments.
- conversion.py  unit/value conversions (e.g., ftm, FC).
- login.py  basic username/password input flow.
- myfirstprogram.py  hello world style starter script.
- mysecondprogram.py  follow-up sample with slightly more logic.

## Requirements
- Python 3.8+

## Run
`
python <file_name>.py
`

## Notes
- These scripts are independent. Run each directly.
- Adjust input prompts per your environment if needed.
